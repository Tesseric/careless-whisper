# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Push-to-talk voice-to-text via local whisper.cpp transcription
- Configurable global hotkey with presets (Option+`, F5, Ctrl+Shift+Space, F19)
- Three English Whisper model sizes: Tiny (75 MB), Base (142 MB), Small (466 MB)
- Smart text injection: Kitty IPC, Accessibility API, clipboard paste fallback
- Terminal-aware floating HUD with live git context dashboard
- Agent overlay system for AI-powered HTML widgets with parameterized templates
- Voice Activity Detection with real-time chunk transcription
- Input device selection with multiple microphone support
- Optional auto-Enter after transcription
- Completion sound toggle
- Launch at login support
- Homebrew cask distribution with code signing and notarization
